### 事件

`trigger-click` 点击文件上传时触发的事件
`preview` 点击预览时触发的事件
`download-file` 点击下载时触发的事件
`download-all` 点击全部下载时触发的事件
`re-upload` 点击重新上传时触发的事件
`click-file-list` 点击上传的文件列表时触发的事件
