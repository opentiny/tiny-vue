### 组件级空态

通过添加`component-empty`属性展示组件级空态，其中 type 类型有`noperm、 nodata、 weaknet、noresult、 nonews`，

:::tip
对应场景：
`noperm` ：无访问权限
`nodata` ：暂无数据
`weaknet` ：网络不给力
`noresult`：无相关搜索结果
`nonews`：暂无最新消息
:::
