### 插槽

:::tip 插槽说明

- header: 头部插槽，默认显示头部，设置 show-header="true" 时有效；
- header-left: 头部左侧插槽，默认显示搜索功能；
- header-right: 头部右侧插槽，默认显示关闭功能；
- footer: 头部插槽，默认隐藏底部，设置 show-footer="true" 时有效；
  :::
