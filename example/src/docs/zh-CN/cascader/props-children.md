## Cascader 级联选择器

<nova-uxlink widget-name="Cascader"></nova-uxlink>

当一个数据集合有清晰的层级结构时，可通过级联选择器逐级查看并选择。

</div>

### 指定选项

:::tip Props

通过 `children ` 指定选项的子选项，默认为 children 。
通过 `value` 指定指定选项的 value 值，默认为 value 。
通过 `label` 指定选项标签,默认为 label 。

:::
