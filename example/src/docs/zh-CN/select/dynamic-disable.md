## Select 选择器

<nova-uxlink widget-name="Dropdown"></nova-uxlink>

Select 选择器是一种通过点击弹出下拉列表展示数据并进行选择的 UI 组件。

</div>

### 禁用

设置 `disabled` 属性后整个选择器不可用。

<nova-demo-view link="select/disabled"></nova-demo-view>

tiny-options 标签设置 `disabled` 属性后该选项不可用。
